# توثيق الصفحات — منصة مشاريع التخرج

هذا الملف يشرح **كل صفحة وظيفية** في المشروع، **ما تحتويه**، و**اللغات والتقنيات** المستخدمة على مستوى المشروع ثم لكل ملف.

---

## اللغات والتقنيات (عام)

| الطبقة | التقنية | الاستخدام |
|--------|---------|------------|
| خادم | **PHP** (≈ 8.2 في Docker / Railway) | توليد HTML، جلسات، نماذج، رفع ملفات، `mysqli` |
| قاعدة البيانات | **MySQL / MariaDB** | جداول: `users`, `projects`, `admins`, `chat_messages` |
| استعلامات | **SQL** | عبر `mysqli` في ملفات `.php` |
| واجهة | **HTML5** | `lang="ar"` و `dir="rtl"` في أغلب الصفحات |
| تنسيق | **CSS3** | ملف مركزي [`style.css`](style.css) + أنماط `<style>` داخلية في بعض الصفحات |
| سكربت المتصفح | **JavaScript** (بدون إطار عمل) | في [`index.php`](index.php): قائمة الجوال، مودال «تواصل معنا»، دردشة مع `fetch` |
| رسوم بيانية | **Chart.js** (CDN) | في [`admin.php`](admin.php) |
| أيقونات | **Font Awesome 6** (CDN) | في [`register.php`](register.php) |
| تخزين كلمات المرور | **bcrypt** | `password_hash` / `password_verify` في PHP |
| نشر (اختياري) | **Docker** | [`Dockerfile`](Dockerfile) — PHP CLI + خادم PHP مدمج |

**ملفات داعمة (ليست صفحات كاملة):**

| الملف | الدور |
|--------|--------|
| [`db_connect.php`](db_connect.php) | اتصال MySQL (محلي أو Railway عبر متغيرات البيئة / `MYSQL_URL`) |
| [`helpers.php`](helpers.php) | دوال مسارات الصور والملفات (محلي أو رابط `https://`) |
| [`admin_side_bar.php`](admin_side_bar.php) | شريط جانبي يُضمَّن في صفحات الإدارة |

## شرح تفصيلي لكل صفحة

### 1. `index.php` — الرئيسية

- **المحتوى:** شعار، عنوان الموقع، تنقّل (دخول طلاب/إدارة أو ملف المستخدم)، قسم ترويجي مع بحث (`q`) وتصفية قسم (`dept`)، إحصاءات من قاعدة البيانات، بطاقات مشاريع مع صورة ووصف مختصر ورابط للتفاصيل، ترقيم صفحات، تذييل بثلاثة أعمدة.
- **تقنيات إضافية:** JavaScript داخل الصفحة (مودال، دردشة، `fetch` إلى `chat_api.php`)، CSS مضمّن للهيرو والدردشة والمودال.
- **قاعدة البيانات:** قراءة من `projects` و `users` (عدّ المشاريع والطلاب).

### 2. `project_details.php` — تفاصيل مشروع

- **المحتوى:** جلب مشروع بالمعرّف `id` من الرابط؛ عرض صورة (عبر `helpers.php` لدعم `uploads/` أو URL)، ملخص، وسوم تقنيات، بوستر، أزرار PDF.
- **الجلسة:** يُسمح للزائر بعرض التفاصيل؛ منطق الجلسة موجود للتحقق من نوع المستخدم في أجزاء أخرى حسب الكود.
- **قاعدة البيانات:** `SELECT` على `projects`.

### 3. `register.php` — تسجيل طالب جديد

- **المحتوى:** نموذج متعدد الحقول، Checkbox للشروط، رسائل نجاح/خطأ.
- **تقنيات:** Font Awesome من CDN.
- **قاعدة البيانات:** `INSERT` في `users` بعد التحقق من البريد و`password_hash`.

### 4. `login_user.php` — دخول الطالب

- **المحتوى:** بريد + كلمة مرور، `password_verify`.
- **قاعدة البيانات:** `SELECT` من `users`.

### 5. `forgot_password.php` — استعادة كلمة المرور

- **المحتوى:** بريد + كلمة مرور جديدة؛ تحديث مباشر (بدون رابط بريد إلكتروني).
- **قاعدة البيانات:** `UPDATE users`.

### 6. `profile.php` — الملف الشخصي

- **المحتوى:** عرض وتعديل الاسم والـ bio، رفع صورة للمجلد `uploads/profiles/`، تغيير اختياري لكلمة المرور.
- **الصلاحيات:** يتطلب `$_SESSION['user_id']`.
- **قاعدة البيانات:** `UPDATE users`.

### 7. `logout_user.php`

- **المحتوى:** تدمير الجلسة أو أجزائها وإعادة التوجيه (يُستخدم أيضًا من لوحة الإدارة في الشريط الجانبي).

### 8. `login_admin.php` — دخول الإدارة

- **المحتوى:** اسم مستخدم + كلمة مرور للجدول `admins`.
- **قاعدة البيانات:** `SELECT` من `admins` مع `password_verify`.

### 9. `admin.php` — لوحة التحكم الرئيسية

- **المحتوى:** إحصائيات إجمالية، مخططان دائريان/شريطيان بـ Chart.js من بيانات مجمّعة SQL.
- **الصلاحيات:** `admin_logged_in`.
- **تقنيات:** Chart.js من jsDelivr.

### 10. `add_project.php` — إضافة مشروع

- **المحتوى:** حقول عنوان، وصف، قسم، سنة تخرج، تقنيات؛ رفع صورة مشروع، بوستر، ملفات PDF؛ حفظ في `projects`.
- **الصلاحيات:** إدارة فقط.
- **قاعدة البيانات:** `INSERT` + ملفات تحت `uploads/`.

### 11. `edit_project.php` — تعديل مشروع

- **المحتوى:** نفس الحقول مع قيم حالية، معاينة صور/روابط PDF، رفع يستبدل الملفات المحلية فقط (الروابط الخارجية لا تُحذف من القرص).
- **قاعدة البيانات:** `UPDATE projects`.

### 12. `manage_projects.php` — إدارة المشاريع

- **المحتوى:** جدول بقائمة المشاريع، روابط لـ `edit_project.php?id=` و `delete_project.php?id=`.
- **قاعدة البيانات:** `SELECT` من `projects`.

### 13. `delete_project.php` — حذف مشروع

- **المحتوى:** حذف سجلات مرتبطة من القرص ثم `DELETE` من القاعدة (مع استثناء الملفات إذا كانت روابط `http`/`https`).
- **قاعدة البيانات:** `DELETE` من `projects`.

### 14. `manage_students.php` — إدارة الطلاب

- **المحتوى:** جدول بالطلاب وصور مصغّرة.
- **قاعدة البيانات:** `SELECT` من `users`.

### 15. `delete_student.php` — حذف طالب

- **المحتوى:** حذف صورة الملف الشخصي من القرص إن وُجدت ثم حذف السجل.
- **قاعدة البيانات:** `DELETE` من `users`.

### 16. `chat_api.php` — واجهة الدردشة

- **المحتوى:** لا يعرض HTML للمستخدم؛ يرد بـ JSON أو ينفذ INSERT.
- **قاعدة البيانات:** `SELECT` / `INSERT` على `chat_messages`.

## ملخص سريع للغات

| اللغة / التقنية | أين تُستخدم بشكل أساسي |
|-----------------|-------------------------|
| **PHP** | كل ملفات `.php` |
| **SQL** | داخل استعلامات النصوص في PHP |
| **HTML** | مخرجات كل الصفحات |
| **CSS** | `style.css` + أنماط داخلية في `index`, `admin`, `profile`, … |
| **JavaScript** | بشكل رئيسي `index.php` (دردشة، مودال، قائمة) |
| **JSON** | استجابة `chat_api.php` واستهلاكها بـ `fetch` |
